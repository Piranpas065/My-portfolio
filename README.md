# Piranya Paskaran - Portfolio (Flask)

A personal portfolio website built with Flask and static front-end assets.

## Setup

```bash
pip install -r requirements.txt
python api/index.py
```

Open http://localhost:5000

## Deployment

This project is configured for Vercel via `vercel.json`.

## Project Structure

- `api/index.py` - Flask app entry point
- `templates/index.html` - Main page template
- `static/css/style.css` - Stylesheet
- `static/js/main.js` - Front-end interactions
